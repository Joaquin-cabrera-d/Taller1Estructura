# Taller1Estructura

Para correctamente compilar y ejecutar el código
1) Vaya al directorio del programa y copie su ubicación
2) En la CMD, coloque cd [la direccion que acaba de copiar]
3) Coloque el siguiente texto en la CMD "g++ -g Main.cpp classes/Lista.cpp classes/Cancion.cpp -o Main.exe"
4) Dirijase al directorio y ejecute el .exe
