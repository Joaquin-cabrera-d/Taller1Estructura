## Sobre el proyecto
El proyecto consiste en un reproductor de música escrito en C++ utilizando nodos y listas enlazadas que sea ejecutable desde cmd, pueda escribir sobre un .txt y funcione en base a los procedimientos solicitados en el enunciado.

## Ejecución y compilación de Código
Para correctamente compilar y ejecutar el código
1) Vaya al directorio del programa y copie su ubicación
2) En la CMD, coloque cd [la direccion que acaba de copiar]
3) Coloque el siguiente texto en la CMD "g++ -g Main.cpp classes/Lista.cpp classes/Cancion.cpp -o Main.exe"
4) Dirijase al directorio y ejecute el .exe

## Funcionamiento de aplicación
La aplicación funciona mediante las opciones demostradas en pantalla durante la ejecución, utilizando submenus para las acciones que desee realizar el usuario. Hay control de errores para evitar inputs no deseados, asi como control de error para archivos
